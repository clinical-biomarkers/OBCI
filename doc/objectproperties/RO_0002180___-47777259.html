  • 'has curation status' http://purl.obolibrary.org/obo/IAO_0000125
  • definition "w 'has component' p if w 'has part' p and w is such that it can be directly disassembled into into n parts p, p2, p3, ..., pn, where these parts are of similar type."
  • 'editor note' "The definition of 'has component' is still under discussion. The challenge is in providing a definition that does not imply transitivity."
  • 'curator note' "For use in recording has_part with a cardinality constraint, because OWL does not permit cardinality constraints to be used in combination with transitive object properties.
